Chess 1.0
Este é um projeto em Java que implementa um jogo de xadrez com todas as suas regras e funcionalidades. Ele oferece uma interface de usuário simples em console, onde os jogadores podem jogar, movimentar peças, capturar, e identificar situações de xeque e xeque-mate. Recursos especiais como en passant e roque são suportados, proporcionando uma experiência autêntica de xadrez. O jogo também mantém um registro das peças capturadas por cada jogador, oferecendo uma experiência completa e desafiadora para entusiastas do xadrez de todos os níveis.
